Detailed Description

template<typename Tfloat>
struct pr::bspline_basis_functions< Tfloat >

B-spline basis functions.

This helper-structure represents the B-spline basis functions for a specific degree \( d \) and knot sequence \( \{ u_k \} \) where \( k = 0, 1, \ldots, m - 1 \).

Moreover, this structure is an interface for accessing the knot vector, and is used internally by the bspline_curve and bspline_patch classes to compute the B-spline basis functions \( B_{i,j} \) efficiently. It is important to note that this structure does not manage any memory, and should not be directly constructed by client code.

◆ set_knots_open()

template<typename Tfloat>
void pr::bspline_basis_functions< Tfloat >::set_knots_open ( )
inline

Set knots open.

Assuming 0-based indexing,

  • set knots \( u_k = 0 \) for \( k \in [0, d) \),
  • set knots \( u_k = 1 \) for \( k \in [n, m) \),
  • set knots \( u_k = (k - d) / (n - d) \) for \( k \in [d, n) \).

This forces the spline to touch the first and last control points at \( t = 0 \) and \( t = 1 \) respectively. However, the spline is not differentiable at these points.

◆ set_knots_closed()

template<typename Tfloat>
void pr::bspline_basis_functions< Tfloat >::set_knots_closed ( )
inline

Set knots closed.

Assuming 0-based indexing, set knots \( u_k = (k - d) / (n - d) \) for \( k \in [0, m) \). Then, the first \( d \) knots and the last \( d \) knots are outside the standard \( [0, 1] \) domain. That is,

  • knots \( u_k < 0 \) for \( k \in [0, d) \),
  • knots \( u_k > 1 \) for \( k \in (n, m) \).

This forces the spline to be properly differentiable. However, the spline never touches the first and last control points. This is still useful for closed curves, because control points may be repeated to ensure continuity at \( t = 0 \) and \( t = 1 \).

Note
To form a closed curve, it is necessary to (cyclically) repeat control points \( d \) times.
