Cumulative distribution function.

\[ F(x) = \left(x_{j + 1} - x_j\right) \left[\frac{1}{2} \left(f_{j + 1} - f_j\right) t + f_j\right] t + F_j \]

where

\[ t = \frac{x - x_j} {x_{j + 1} - x_j} \]

Exceptions
std::runtime_errorUnless points_.size() > 1.

◆ cdfinv()

template<typename T = double>
float_type pre::piecewise_linear_distribution< T >::cdfinv ( float_type  u) const
inline

Cumulative distribution function inverse.

\[ F^{-1}(u) = (1 - t) x_j + t x_{j + 1} \]

where

\[ t \in [0, 1) \implies a_2 t^2 + a_1 t + a_0 = u \]

where, in turn,

  • \( a_2 \gets (x_{j + 1} - x_j) (f_{j + 1} - f_j) / 2 \)
  • \( a_1 \gets (x_{j + 1} - x_j) f_j \)
  • \( a_0 \gets F_j \)
Exceptions
std::runtime_errorUnless points_.size() > 1.
