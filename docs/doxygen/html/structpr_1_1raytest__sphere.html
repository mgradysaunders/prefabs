Detailed Description

template<typename T>
struct pr::raytest_sphere< T >

Ray-test (sphere).

This implementation is based on the implementation of Pharr, Humphreys, and Jakob in Physically based rendering: from theory to implementation (3rd edition).

Surface area probability density function sampling routine.

  • \( z \gets (1 - u_{[1]})z_{\min} + u_{[1]}z_{\max} \)
  • \( \theta \gets \arccos(z / r) \)
  • \( \phi \gets u_{[0]} \phi_{\max} \)

    \[ \mathbf{p}_{\text{hit}} = \begin{bmatrix} r \sin(\theta) \cos(\phi) \\ r \sin(\theta) \sin(\phi) \\ r \cos(\theta) \end{bmatrix} \]
