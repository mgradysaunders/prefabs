Wrap Ttraits::abs().

For every \( x \in T \), the implementation must satisfy

  • \( |x| = \sqrt{x^\dagger x} \in \mathbb{R} \),
  • \( |x| \ge 0 \), and
  • \( |x| = 0 \) iff \( x \sim 0 \).

int  q,
dense_matrix_view< value_type *>  x,
dense_matrix_view< value_type *>  y = {} 
)
inlinestatic

Householder step.

For \( \mathbf{X} \in T^{m \times n} \), construct the \( (p, q) \) reflector \( \mathbf{w} \in T^{m-p} \) as follows:

  • let \( \mathbf{w} = X_{[p:m,q]} \);
  • let \( \alpha = \operatorname{sign}(w_{[0]}) \lVert\mathbf{w}\rVert \);
  • update \( \mathbf{w} \gets \mathbf{w} + \alpha\mathbf{\delta}_0 \);
  • update \( \mathbf{w} \gets \mathbf{w} / \lVert\mathbf{w}\rVert \) to normalize.

For \( j \ge q \), reflect the \( X_{[p:m,j]} \) over \( \mathbf{w} \). This assumes sequential application of Householder steps, such that \( X_{[p:m,j]} = 0 \) for \( j < q \).

Parameters
[in]pTarget index \( p \).
[in]qTarget index \( q \).
[in,out]xMatrix \( \mathbf{X} \).
[in,out]yMatrix \( \mathbf{Y} \). Optional.
Note
This method maintains a static thread-local buffer of temporary values. If x is empty, the implementation resets the buffer and returns immediately.
