\( L^2 \) length, fast variant.

Expression

\[ \lVert\mathbf{x}\rVert = \sqrt{\sum_k |x_{[k]}|^2} \]

Note
Fast means the implementation
  • calculates the sum of square moduli directly, implicitly assuming the square terms neither overflow nor underflow, and the sum of square terms does not overflow.

\( L^2 \) length, safe variant.

Expression

\[ \lVert\mathbf{x}\rVert = \sqrt{\sum_k |x_{[k]}|^2} \]

Note
Safe means the implementation
  • calculates the moduli as a preprocessing step,
  • if impending overflow or underflow, factors the maximum modulus out from under the radical.

\( L^2 \) normalize, fast variant.

Expression

\[ \hat{\mathbf{x}} = \frac{\mathbf{x}}{\lVert\mathbf{x}\rVert} \]

Note
Fast means the implementation
  • calls length_fast() to calculate length,
  • multiplies by reciprocal of length, implicitly assuming reciprocal does not overflow.

◆ normalize_safe()

inline

\( L^2 \) normalize, safe variant.

Expression

\[ \hat{\mathbf{x}} = \frac{\mathbf{x}}{\lVert\mathbf{x}\rVert} \]

Note
Safe means the implementation
  • calls length_safe() to calculate length,
  • if length is zero, returns all zeros,
  • if length is less than the minimum invertible floating point value, divides by length instead of multiplying by reciprocal.
