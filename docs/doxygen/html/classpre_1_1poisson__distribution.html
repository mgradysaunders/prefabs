Detailed Description

template<typename T = double>
class pre::poisson_distribution< T >

Poisson distribution.

Note
This implementation is not tenable for large \( \lambda \) (say \( \lambda > 20 \)), due to the direct calculation of the cumulative distribution function and its inverse. For \( \lambda > 20 \), it is preferable to approximate the Poisson distribution \( P(\lambda) \) with the binomial distribution \( B(n, p) \) by choosing \( p \approx 0.25 \), \( n = \lambda / p \).

Cumulative distribution function.

\[ F(x) = e^{-\lambda} \sum_{j=0}^{\lfloor x \rfloor}\frac{\lambda^j}{j!} \]

Note
This implementation computes \( F \) directly. Thus, the time complexity is \( O(\lfloor x \rfloor) \).
