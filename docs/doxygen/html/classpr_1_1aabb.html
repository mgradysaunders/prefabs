◆ volume()

template<typename T, std::size_t N>
constexpr T pr::aabb< T, N >::volume ( ) const
inline

Hyper volume.

Note
Hyper volume in \( N \) dimensions measures the entire \( N \) dimensional subspace. That is,

\[ V = \prod_{k \in L} (P_{\max[k]} - P_{\min[k]}) \]

for index set \( L = [0, N) \).
  • In \( N = 2 \) dimensions, \( V \) measures signed area.
  • In \( N = 3 \) dimensions, \( V \) measures signed volume.

◆ surface_area()

constexpr T pr::aabb< T, N >::surface_area ( ) const

Hyper surface area.

Note
Hyper surface area in \( N \) dimensions measures the \( N - 1 \) dimensional subspace on the border of the \( N \) dimensional volume. That is,

\[ S = \sum_{k \in L} \prod_{j \in L_k} 2 (P_{\max[j]} - P_{\min[j]}) \]

for index sets \( L = [0, N) \) and \( L_k = L \setminus k \).
  • In \( N = 2 \) dimensions, \( S \) measures signed perimeter.
  • In \( N = 3 \) dimensions, \( S \) measures signed surface area.
