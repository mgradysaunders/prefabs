Detailed Description

template<typename Tfloat>
struct pr::oren_nayar_diffuse< Tfloat >

Oren-Nayar diffuse BRDF.

Oren-Nayar diffuse BRDF, described by M. Oren and S. K. Nayar in “Generalization of Lambert’s reflectance model” in 1994.
