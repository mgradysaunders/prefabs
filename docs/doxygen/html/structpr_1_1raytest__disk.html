template<typename T>

Ray-test (disk).

This implementation is based on the implementation of Pharr, Humphreys, and Jakob in Physically based rendering: from theory to implementation (3rd edition).

template<typename T >
hit_type pr::raytest_disk< T >::surface_area_pdf_sample ( multi< float_type, 2 >  u) const
inline

Surface area probability density function sampling routine.

  • \( r \gets \sqrt{(1 - u_{[0]}) r_{\min}^2 + u_{[0]} r_{\max}^2} \)
  • \( \phi \gets u_{[1]} \phi_{\max} \)

    \[ \mathbf{p}_{\text{hit}}(\mathbf{u}) = \begin{bmatrix} r \cos{\phi} \\ r \sin{\phi} \\ h \end{bmatrix} \]

Parameters
[in]uSample in \( [0, 1)^2 \).
Note
If rmin_ == 0 and phimax_ == 2 * pi, the implementation uses the concentric disk sampling routine uniform_disk_pdf_sample().

Solid angle probability density function.

  • \( \mathbf{v}_i \gets \mathbf{p}_{\text{hit}} - \mathbf{p}_{\text{ref}} \)
  • \( \omega_i \gets \normalize(\mathbf{v}_i) \)
  • \( \omega_g \gets [0\; 0\; 1]^\top \)

    \[ f_{\omega}( \mathbf{p}_{\text{ref}} \to \mathbf{p}_{\text{hit}}) = \frac{1}{A} \frac{|\mathbf{v}_i \cdot \mathbf{v}_i|} {|\omega_i \cdot \omega_g|} = \frac{1}{A} \frac{\lVert \mathbf{v}_i \rVert} {|\mathbf{v}_{i[2]}|} \]

Parameters
[in]prefReference point.
[in]phitHit point.
Note
For efficiency, the implementation assumes \( \mathbf{p}_{\text{hit}} \) is actually on the surface.
