Detailed Description

template<typename T, typename Tstate, Tstate Nmultiplier, Tstate Ndefault_increment>
class pre::pcg_xsh_rr_engine< T, Tstate, Nmultiplier, Ndefault_increment >

PCG XSH-RR engine.

Permuted congruential generator with XOR-shift plus random-rotate output transform. The implementation here is adapted as a special case of the implementation by Melissa O'Neill affiliated with the PCG project.
