Detailed Description

template<typename T, typename Talloc = std::allocator<T>>
class pr::block_array2< T, Talloc >

2-dimensional block array.

2-dimensional block array with block size \( B = 2^k \) for an integer \( k > 0 \), which is chosen to promote cache performance. Specifically, \( k \) is chosen such that each \( B^2 \) block fits on an L1 cache line which, by default, the implementation assumes to be 64 bytes. To change this assumption, define L1_LINE to the appropriate size before including the data structure.

2-dimensional index to 1-dimensional index.

2-dimensional index \( (l_0, l_1) \) to 1-dimensional index \( j \). In particular,

\[ j = B^2 (q_0 + (n_0 / B) q_1) + (r_0 + B r_1) \]

where the \( q_k, r_k \) denote the quotients and remainders of the \( l_k / B \) respectively.

multi<size_type, 2> pr::block_array2< T, Talloc >::convert ( size_type  pos) const
inlinenoexcept

1-dimensional index to 2-dimensional index.

1-dimensional index \( j \) to 2-dimensional index \( (l_0, l_1) \). In particular,

\[ l_k = B q_k + r_k \]

where

  • \( q_0 \) and \( q_1 \) denote the remainder and quotient of \( q / (n_0 / B) \) respectively, and
  • \( r_0 \) and \( r_1 \) denote the remainder and quotient of \( r / B \) respectively,

where, in turn, \( q \) and \( r \) denote the quotient and remainder of \( j / B^2 \) respectively.
