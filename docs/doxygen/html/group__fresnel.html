Fresnel equations for dielectric interface.

Parameters
[in]etaRefractive index \( \eta = \eta_i / \eta_t \).
[in]cos_thetaiCosine of incidence angle.
[out]cos_thetatCosine of transmittance angle.
[out]rsReflection s-polarization.
[out]rpReflection p-polarization.
[out]tsTransmission s-polarization.
[out]tpTransmission p-polarization.
Expression
  • \( \cos^2{\theta_t} = 1 - \eta^2 (1 - \cos^2{\theta_i}) \)
  • \( r_s = (\eta \cos{\theta_i} - \cos{\theta_t}) / (\eta \cos{\theta_i} + \cos{\theta_t}) \)
  • \( r_p = (\cos{\theta_i} - \eta \cos{\theta_t}) / (\cos{\theta_i} + \eta \cos{\theta_t}) \)
  • \( t_s = 1 + r_s \)
  • \( t_p = \eta (1 + r_p) \)
Note
In the case of total internal reflection,
  • cos_thetat = 0,
  • rs = rp = 1, and
  • ts = tp = 0.
Returns
If total internal reflection (TIR), returns false to indicate that there is no refracted ray. Otherwise, returns true.

Parameters

[out]cos_thetatCosine of transmittance angle.

Expression
  • \( R_0 = (\eta - 1)^2 / (\eta + 1)^2 \)
  • \( F_r = R_0 + (1 - R_0) (1 - \cos{\theta_i})^5 \)
  • \( F_t = 1 - F_r \)
Returns
If total internal reflection (TIR), returns false to indicate that there is no refracted ray. Otherwise, returns true.
