Fresnel equations for dielectric-dielectric interface.

  • \( \cos^2{\theta_t} = 1 - n^2 (1 - \cos^2{\theta_i}) \)
  • \( r_s = (n \cos{\theta_i} - \cos{\theta_t}) / (n \cos{\theta_i} + \cos{\theta_t}) \)
  • \( r_p = (\cos{\theta_i} - n \cos{\theta_t}) / (\cos{\theta_i} + n \cos{\theta_t}) \)
  • \( t_s = 1 + r_s \)
  • \( t_p = n (1 + r_p) \)
Parameters
[in]cos_thetaiCosine of incidence angle.

[in]n1Refractive index in lower hemisphere.

[out]rpReflection p-polarization.
[out]tsTransmission s-polarization.
[out]tpTransmission p-polarization.
Returns
Cosine of transmittance angle.
Note
In the event of total internal reflection,
  • sets rs = rp = 1,
  • sets ts = tp = 0, and
  • returns 0.
