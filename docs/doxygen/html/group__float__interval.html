Detailed Description

<preform/float_interval.hpp>

C++ version: >=C++17

Note
To use std::fesetround instead of pre::fdec and pre::finc where sensible, #define PREFORM_USE_FENV 1 before including. This ought to yield tighter bounds, though the code may not run any faster.
If PREFORM_USE_FENV is truthy, this header will #pragma STDC FENV_ACCESS ON to be standard compliant. GCC and clang may warn that this pragma is unknown—this is not an issue, as GCC and clang (on x86-64 at least) provide floating-point environment access by default.
