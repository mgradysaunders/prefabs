◆ cdf()

template<typename T = double, typename Talloc = std::allocator<T>>
T pr::piecewise_constant_distribution< T, Talloc >::cdf ( int  k) const
inline

Cumulative density function.

\[ c(k) = \begin{cases} c_{[k - 1]} & k \in [1, n] \\ 0 & k \notin [1, n], k < 1 \\ 1 & k \notin [1, n], k > n \end{cases} \]

Note
This follows the convention that \( c(k) = P(X < k) \). Hence, \( c(0) = 0 \) even if \( p(0) > 0 \).
