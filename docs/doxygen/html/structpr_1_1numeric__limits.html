For floating point types, minimum invertible value.

The minimum invertible value \( \varepsilon_{\text{inv}} \) is the minimum positive value satisfying \( 1 \oslash x < \infty \).

For IEEE floating point types (and possibly others?), \( \varepsilon_{\text{inv}} = \varepsilon_{\text{min}} / 4 + \Delta_{\text{denormal}} \) where \( \varepsilon_{\text{min}} \) is the minimum positive normal value and \( \Delta_{\text{denormal}} \) is the denormal spacing.

For floating point types, machine epsilon.

Machine epsilon \( \varepsilon_{\mathrm{m}} \), also known as unit roundoff, is the maximum representable value satisfying \( 1 \oplus x = 1 \).

For IEEE floating point types (and possibly others?), \( \varepsilon_{\text{m}} = \varepsilon_{\text{std}} / 2 \) where \( \varepsilon_{\text{std}} \) is standard epsilon, the difference between 1 and the next representable value.
