T  n 
)

Wrap integer in range.

  • If \( n > 0 \), wrap \( k \) to \( [0, n) \).
  • If \( n < 0 \), wrap \( k \) to \( (-n, 0] \).
Note
For \( k > 0 \), \( n > 0 \), this operation is equivalent to \( k \% n \).

Parameters

[in]nStep count.
Note
