Phase function.

\[ p_s(\omega_o, \omega_i) = \frac{3}{16\pi} \left\lbrack{ \frac{1 - \gamma}{1 + 2\gamma} (\omega_o\cdot\omega_i)^2 + \frac{1 + 3\gamma}{1 + 2\gamma} }\right\rbrack \]

where

\[ \gamma = \frac{\rho}{2 - \rho} \]

Parameters
[in]woOutgoing direction.
[in]wiIncident direction.
