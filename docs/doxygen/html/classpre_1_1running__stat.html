running_stat pre::running_stat< T >::operator+ ( T  x) const
inline

Add term.

Note
  • \( n' \gets n + 1 \)
  • \( m' \gets (n m + x) / n' \)
  • \( s \gets (x - m) / n' \)
  • \( t \gets (x - m)^2 n / n' \)
  • \( M_2' \gets M_2 + t \)
  • \( M_3' \gets M_3 + s t (n - 1) + 3 s M_2 \)
  • \( M_4' \gets M_4 + s^2 t (n^2 - n + 1) + 6 s^2 M_2 - 4 s M_3 \)

◆ operator+() [3/3]

template<typename T = double>
running_stat pre::running_stat< T >::operator+ ( const running_stat< T > &  b) const
inline

Add terms.

Note
  • \( n' \gets n_A + n_B \)
  • \( r_A \gets n_A / n' \)
  • \( r_B \gets n_B / n' \)
  • \( d \gets m_B - m_A \)
  • \( p \gets n_A n_B / n' \)
  • \( q \gets d^2 p \)
  • \( m' \gets m_A + r_B d \)
  • \( M_2' \gets M_{2,A} + M_{2,B} + q \)
  • \( M_3' \gets M_{3,A} + M_{3,B} + q d (r_A - r_B) + 3 d (r_A M_{2,B} - r_B M_{2,A}) \)
  • \( M_4' \gets M_{4,A} + M_{4,B} + q d^2 (r_A^2 - r_A r_B + r_B^2) + 6 d^2 (r_A^2 M_{2,B} - r_B^2 M_{2,A}) + 4 d (r_A M_{3,B} - r_B M_{3,A}) \)
