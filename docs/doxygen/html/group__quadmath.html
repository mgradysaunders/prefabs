◆ sign() [1/2]

__float128 pre::sign ( __float128  x)
inline

Sign function.

\[ \operatorname{sign}(x) = \begin{cases} -1 & x < 0 \\ +1 & x > 0 \end{cases} \]

Note
Uses copysignq(). Hence,
  • pre::sign(-0.0) = -1.0 and
  • pre::sign(+0.0) = +1.0.

◆ step() [1/2]

__float128 pre::step ( __float128  x)
inline

Step function.

\[ \operatorname{step}(x) = \frac{1}{2} \operatorname{sign}(x) + \frac{1}{2} = \begin{cases} 0 & x < 0 \\ 1 & x > 0 \end{cases} \]

Note
Uses signbitq(). Hence,
  • pre::step(-0.0) = 0.0 and
  • pre::step(+0.0) = 1.0.

◆ sign() [2/2]

__complex128 pre::sign ( const __complex128 &  x)
inline

Sign function.

\[ \operatorname{sign}(x) = \begin{cases} 1 & x = 0 \\ x/|x| & x \ne 0 \end{cases} \]

Note
If pre::imag(x) == 0, computes pre::sign(pre::real(x)) and preserves sign of pre::imag(x).

◆ step() [2/2]

__complex128 pre::step ( const __complex128 &  x)
inline

Step function.

\[ \operatorname{step}(x) = \frac{1}{2} \operatorname{sign}(x) + \frac{1}{2} \]

Note
If pre::imag(x) == 0, computes pre::step(pre::real(x)) and preserves sign of pre::imag(x).
