Phase function.

\[ p_s(\omega_o, \omega_i) = \sum_k \frac{1}{4\pi} \frac{1 - g_k^2} {(1 + g_k^2 - 2g_k\omega_o\cdot\omega_i)^{3/2}} w_k \]

[in]wiIncident direction.
