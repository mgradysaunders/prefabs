Single-scattering BSDF.

Parameters
[in]woOutgoing direction.
[in]wiIncident direction.
Expression

If \( \omega_{o_z} < 0 \), flip everything:

  • \( \omega_{o_z} \gets -\omega_{o_z} \)
  • \( \omega_{i_z} \gets -\omega_{i_z} \)
  • \( \eta \gets 1 / \eta \)

If \( \omega_{i_z} > 0 \), calculate the BRDF:

  • \( \mathbf{v}_m \gets \omega_o + \omega_i \)
  • \( \omega_m \gets \mathbf{v}_m / \lVert \mathbf{v}_m \rVert \)

  • \( G_2(\omega_o, \omega_i) = 1 / (1 + \Lambda(\omega_o) + \Lambda(\omega_i)) \)

    \[ f_s(\omega_o \to \omega_i) = \frac{1}{\omega_{o_z}} D(\omega_m) F_r(\omega_o \cdot \omega_m) G_2(\omega_o, \omega_i) \]

If \( \omega_{i_z} < 0 \), calculate the BTDF:

  • \( \mathbf{v}_m \gets \eta\omega_o + \omega_i \)
  • \( \mathbf{v}_m \gets -\mathbf{v}_m \) if \( v_{m_z} < 0 \)
  • \( \omega_m \gets \mathbf{v}_m / \lVert \mathbf{v}_m \rVert \)

  • \( G_2(\omega_o, \omega_i) = \beta(1 + \Lambda(\omega_o), 1 + \Lambda(\omega_i)) \)

    \[ f_s(\omega_o \to \omega_i) = \frac{1}{\omega_{o_z}} D(\omega_m) F_t(\omega_o \cdot \omega_m) G_2(\omega_o, \omega_i) |\omega_o \cdot \omega_m| |\omega_i \cdot \omega_m| \frac{1}{\lVert \mathbf{v}_m \rVert^2} \]

Note

\[ f_s(\omega_o \to \omega_i) \frac{1}{|\omega_{i_z}|\eta_i^2} = f_s(\omega_i \to \omega_o) \frac{1}{|\omega_{o_z}|\eta_o^2} \]

Single-scattering BSDF-PDF.

Parameters
[in]woOutgoing direction.
[in]wiIncident direction.
Expression

If \( \omega_{o_z} < 0 \), flip everything:

  • \( \omega_{o_z} \gets -\omega_{o_z} \)
  • \( \omega_{i_z} \gets -\omega_{i_z} \)
  • \( \eta \gets 1 / \eta \)

If \( \omega_{i_z} > 0 \), calculate BRDF density:

  • \( \mathbf{v}_m \gets \omega_o + \omega_i \)

  • \( \omega_m \gets \mathbf{v}_m / \lVert \mathbf{v}_m \rVert \)

    \[ f_{s,\text{bsdf}}(\omega_o \to \omega_i) = D_{\omega_o}(\omega_m) F_r(\omega_o \cdot \omega_m) \frac{1}{4\omega_i \cdot \omega_m} \]

If \( \omega_{i_z} < 0 \), calculate BTDF density:

  • \( \mathbf{v}_m \gets -\eta\omega_o - \omega_i \)
  • \( \mathbf{v}_m \gets -\mathbf{v}_m \) if \( \eta > 1 \)

  • \( \omega_m \gets \mathbf{v}_m / \lVert \mathbf{v}_m \rVert \)

    \[ f_{s,\text{bsdf}}(\omega_o \to \omega_i) = D_{\omega_o}(\omega_m) F_t(\omega_o \cdot \omega_m) \frac{|\omega_i \cdot \omega_m|} {\lVert \mathbf{v}_m \rVert^2} \]
