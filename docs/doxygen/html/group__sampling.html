Uniform disk probability density function sampling routine.

\[ \mathbf{P}_{\text{disk}}(\mathbf{u}) = \begin{bmatrix} r \cos{\theta} \\ r \sin{\theta} \end{bmatrix} \]

where

  • \( u_{[0]}' \gets 2 u_{[0]} - 1 \)
  • If \( |u_{[0]}'| > |u_{[1]}| \):

    \[ (r, \theta) = \left(u_{[0]}', \frac{\pi}{4}\frac{u_{[1]}}{u_{[0]}'}\right) \]

  • If \( |u_{[0]}'| \le |u_{[1]}| \):

    \[ (r, \theta) = \left(u_{[1]}, \frac{\pi}{2} - \frac{\pi}{4}\frac{u_{[0]}'}{u_{[1]}}\right) \]

Parameters
[in]uSample in \( [0, 1)^2 \).

Build 3-dimensional orthonormal basis.

  • \( \alpha_0 \gets -1 / (\hat{z}_{[2]} + 1) \)
  • \( \alpha_1 \gets \alpha_0 \hat{z}_{[0]} \hat{z}_{[1]} \)
  • \( \alpha_2 \gets \alpha_0 \hat{z}_{[0]}^2 + 1 \)
  • \( \alpha_3 \gets \alpha_0 \hat{z}_{[1]}^2 + 1 \)
  • \( \hat{\mathbf{x}} \gets [\alpha_2\; \alpha_1\; -\hat{z}_{[0]}]^\top \)
  • \( \hat{\mathbf{y}} \gets [\alpha_1\; \alpha_3\; -\hat{z}_{[1]}]^\top \)
Note
As the notation suggests, the implementation assumes the input vector hatz is unit-length.
