Detailed Description

template<typename T, std::size_t N>
class pr::static_stack< T, N >

Static stack.

Static stack (first-in last-out), where elements are pushed onto and popped from the top/back.

Template Parameters
TStack value type, which must be
  • default constructible,
  • copy assignable, and
  • destructible.
NStack capacity, which must be positive.
