◆ sign() [1/2]

template<typename T >
auto pr::sign ( T  x) -> decltype(std::copysign(T(1), x))
inline

Sign function.

\[ \operatorname{sign}(x) = \begin{cases} -1 & x < 0 \\ +1 & x > 0 \end{cases} \]

Note
Uses std::copysign(). Hence,
  • pr::sign(-0.0) = -1.0 and
  • pr::sign(+0.0) = +1.0.

◆ step() [1/2]

template<typename T >
auto pr::step ( T  x) -> decltype(std::signbit(x) ? T(0) : T(1))
inline

Step function.

\[ \operatorname{step}(x) = \frac{1}{2} \operatorname{sign}(x) + \frac{1}{2} = \begin{cases} 0 & x < 0 \\ 1 & x > 0 \end{cases} \]

Note
Uses std::signbit(). Hence,
  • pr::step(-0.0) = 0.0 and
  • pr::step(+0.0) = 1.0.

◆ sign() [2/2]

Sign function.

\[ \operatorname{sign}(x) = \begin{cases} 1 & x = 0 \\ x/|x| & x \ne 0 \end{cases} \]

Note
If pr::imag(x) == 0, computes pr::sign(pr::real(x)) and preserves sign of pr::imag(x).

◆ step() [2/2]

Step function.

\[ \operatorname{step}(x) = \frac{1}{2} \operatorname{sign}(x) + \frac{1}{2} \]

◆ betai()

template<typename T >
std::enable_if_t< std::is_floating_point<T>::value, T> pr::betai ( T  x,
T  p,
T  q 
)
inline

Regularized incomplete beta function.

\[ I(x;p,q) = \frac{1}{\beta(p, q)} \int_0^x t^{p-1} (1 - t)^{q-1}\, dt \]

for

  • \( x \in [0, 1] \)
  • \( p \in (0, \infty) \)
  • \( q \in (0, \infty) \)
Exceptions
std::runtime_errorIf implementation fails to converge.
See also
Codeplea's article.

◆ binom()

template<typename T >
std::enable_if_t< std::is_arithmetic<T>::value, T> pr::binom ( T  n,
T  k 
)
inline

Binomial coefficient.

\[ {n \choose k} = \frac{n!}{k!(n - k)!} = \frac{\Gamma(n + 1)}{\Gamma(k + 1)\Gamma(n - k + 1)} \]

Exceptions
std::runtime_errorIf T is integral and the calculation overflows.
Note
If T is floating point, but both n and k are exactly representable as integers, delegates to integral implementation. If, in turn, the integral calculation overflows, returns infinity (and does not throw).
If T is floating point and calculation overflows, returns infinity (and does not throw).
