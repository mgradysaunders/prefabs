◆ sign() [1/2]

template<typename T >
auto pre::sign ( T  x) -> decltype(std::copysign(T(1), x))
inline

Sign function.

\[ \operatorname{sign}(x) = \begin{cases} -1 & x < 0 \\ +1 & x > 0 \end{cases} \]

Note
Uses std::copysign(). Hence,
  • pre::sign(-0.0) = -1.0 and
  • pre::sign(+0.0) = +1.0.

◆ step() [1/2]

template<typename T >
auto pre::step ( T  x) -> decltype(std::signbit(x) ? T(0) : T(1))
inline

Step function.

\[ \operatorname{step}(x) = \frac{1}{2} \operatorname{sign}(x) + \frac{1}{2} = \begin{cases} 0 & x < 0 \\ 1 & x > 0 \end{cases} \]

Note
Uses std::signbit(). Hence,
  • pre::step(-0.0) = 0.0 and
  • pre::step(+0.0) = 1.0.

◆ sign() [2/2]

Sign function.

\[ \operatorname{sign}(x) = \begin{cases} 1 & x = 0 \\ x/|x| & x \ne 0 \end{cases} \]

Note
If pre::imag(x) == 0, computes pre::sign(pre::real(x)) and preserves sign of pre::imag(x).

Step function.

\[ \operatorname{step}(x) = \frac{1}{2} \operatorname{sign}(x) + \frac{1}{2} \]

◆ betai()

template<typename T >

T  p,
T  q 
)
inline

Regularized incomplete beta function.

\[ I(x;p,q) = \frac{1}{\beta(p, q)} \int_0^x t^{p-1} (1 - t)^{q-1}\, dt \]

for

  • \( x \in [0, 1] \)
  • \( p \in (0, \infty) \)
  • \( q \in (0, \infty) \)
Exceptions
std::runtime_errorIf implementation fails to converge.
See also
Codeplea's article.
