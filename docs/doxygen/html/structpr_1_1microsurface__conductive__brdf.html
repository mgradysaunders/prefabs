Single-scattering BRDF.

If \( \omega_{o_z} < 0 \), flip everything:

  • \( \omega_{o_z} \gets -\omega_{o_z} \)
  • \( \omega_{i_z} \gets -\omega_{i_z} \)
  • \( \eta \gets 1 / \eta \)

If \( \imag{\eta} > 0 \), quit:

\[ f_{s}(\omega_o \to \omega_i) = 0 \]

If \( \omega_{i_z} > 0 \), calculate the BRDF:

  • \( \mathbf{v}_m \gets \omega_o + \omega_i \)
  • \( \omega_m \gets \mathbf{v}_m / \lVert \mathbf{v}_m \rVert \)

  • \( G_2(\omega_o, \omega_i) = 1 / (1 + \Lambda(\omega_o) + \Lambda(\omega_i)) \)

    \[ f_s(\omega_o, \omega_i) = \frac{1}{\omega_{o_z}} D(\omega_m) F_r(\omega_o \cdot \omega_m) G_2(\omega_o, \omega_i) \]

[in]wiIncident direction.

Single-scattering BRDF-PDF.

If \( \omega_{o_z} < 0 \), flip everything:

  • \( \omega_{o_z} \gets -\omega_{o_z} \)
  • \( \omega_{i_z} \gets -\omega_{i_z} \)
  • \( \eta \gets 1 / \eta \)

If \( \imag{\eta} > 0 \), quit:

\[ f_{s,\text{brdf}}(\omega_o \to \omega_i) = 0 \]

If \( \omega_{i_z} > 0 \), calculate BRDF density:

  • \( \mathbf{v}_m \gets \omega_o + \omega_i \)
  • \( \omega_m \gets \mathbf{v}_m / \lVert \mathbf{v}_m \rVert \)

    \[ f_{s,\text{brdf}}(\omega_o \to \omega_i) = D_{\omega_o}(\omega_m) \frac{1}{4\omega_i \cdot \omega_m} \]

Parameters
[in]woOutgoing direction.
[in]wiIncident direction.

std::complex<float_type>(float_type(1.5), float_type(4.0))

Refractive index \( \eta \).

\[ \eta = \frac{\eta_{+}} {\eta_{-}} \]

where

  • \( \eta_{+} \) is the refractive index in the upper hemisphere
  • \( \eta_{-} \) is the refractive index in the lower hemisphere

The implementation assumes that either

  • \( \imag{\eta_{+}} > 0 \), \( \imag{\eta_{-}} = 0 \), or
  • \( \imag{\eta_{+}} = 0 \), \( \imag{\eta_{-}} > 0 \).

That is, the implementation assumes the dielectric hemisphere where the BRDF is non-zero is the upper hemisphere if \( \imag{\eta} < 0 \) and the lower hemisphere if \( \imag{\eta} > 0 \).
